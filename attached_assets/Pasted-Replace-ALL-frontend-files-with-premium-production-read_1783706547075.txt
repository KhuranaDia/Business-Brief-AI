frontend/src/hooks/useApi.js:
Custom hook for all API calls:
- useBriefs() — fetches GET /api/briefs, auto-refreshes every 30s
- useGenerateBrief(data) — posts to /api/analyze, returns loading/result/error states
- useUploadCSV(file) — posts to /api/upload-csv

Base URL from env: VITE_API_URL or default http://localhost:8000

All components must:
- Handle loading states gracefully
- Handle error states with user-friendly messages in red
- Be fully responsive (mobile first)
- Use only Tailwind classes
- Have smooth transitions (transition-all duration-200)
- Never use any external UI library — pure Tailwind only
