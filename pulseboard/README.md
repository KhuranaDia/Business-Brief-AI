# PulseBoard

An autonomous multi-agent AI system that analyzes business data and generates a plain-English morning brief.

## Structure

- backend/ — Python + FastAPI, multi-agent pipeline (revenue, behavior, error, sentiment, synthesis)
- frontend/ — React + Tailwind CSS dashboard
- PostgreSQL database
- Fireworks AI (llama-v3p1-70b-instruct) for LLM analysis
